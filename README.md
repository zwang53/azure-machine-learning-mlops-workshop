# Azure Machine Learning in a Day - MLOps

本次实验旨在通过一天的动手实践，帮助大家理解MLOps，并基于微软机器学习服务开展机器学习项目。

## 实验目标

**Goal:**
* 基于微软机器学习实现基本的机器学习项目
* 理解MLOps的基本概念和使用方法
* 学习如何使用微软机器学习服务来实现MLOps
* 重点针对AML Pipeline做丰富的联系
